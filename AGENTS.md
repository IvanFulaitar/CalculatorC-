# AGENTS.md

## Project Overview
- Console calculator project built as a minimal C# .NET 9 application.
- The application should remain simple, readable, and easy to test.

## Tech Stack
- C# with .NET 9
- xUnit for tests

## Build Commands
- `dotnet build`
- `dotnet test`
- `dotnet run`

## Structure
- `src/` for application source code
- `tests/` for test code

## Conventions
- Use PascalCase for classes.
- Do not add external libraries except xUnit.
